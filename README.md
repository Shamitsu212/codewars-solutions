# codewars-solutions

В этом репозитории собраны мои решения задач с платформы [Codewars](https://www.codewars.com/).
[Мой профиль на платформе Codewars](https://www.codewars.com/users/Shamitsu212)

## О репозитории

- Решения задач разной сложности (от **8 kyu** до более высоких уровней)
- Код написан в учебных и практических целях

## Структура

- Задачи разбиты по папкам
- В каждой папке есть файл task.txt с описанием задачи и файл solution.js с решением  
- Каждая задача написана на языке JavaScript
